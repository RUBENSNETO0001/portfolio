Dado Random

Regras

Dados Mentirosos Cada jogador lança os dados e verifica o resultado sem revelar para os outros jogadores. O objetivo é ser o único jogador com um ou mais dados.
General Cada jogador lança os dados e marca uma jogada prevista em sua cartela. Uma vez marcada, aquela jogada não pode ser repetida pelo mesmo jogador até o final da partida.
Dado de apostas O jogador gira o dado para que a face com o número "2" fique voltada para cima, mostrando que o jogo foi dobrado. O oponente tem de escolher entre aceitar a nova aposta ou desistir do jogo.
Banco Imobiliário e RPGs Quando não se tem dois dados com mesmo número de lados, joga-se o mesmo dado duas vezes.
